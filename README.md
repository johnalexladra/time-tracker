# time-tracker
Time Tracker using MERN (MySQL, ExpressJS, ReactJS, NodeJS)


Clone the project using this command in Terminal
`````
git clone https://github.com/johnalexladra/time-tracker.git
``````


To run server:
`````
cd server
npm install
npm start
``````

To run client
`````
cd client
npm install
npm run dev
``````
